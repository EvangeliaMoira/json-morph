# json-morph

A fast, single-file, browser-based utility designed for managing, filtering, and batch-updating JSON datasets. This tool is optimized for developers and translators handling localization files, configuration objects, or structured data arrays without needing external servers or complex command-line tools.

## ✨ Features
*   **Zero Dependencies:** Built entirely with native HTML5, CSS3 variables, and vanilla modern JavaScript. Completely client-side for maximum privacy and speed.
*   **Smart Query Pipelines:** Dynamically filters large JSON data sets by Language metadata or specific Object Keys.
*   **Advanced Find & Replace:** 
    *   Batch-updates string patterns across filtered values.
    *   Toggleable **Key Mutation** to rename object structural properties on the fly.
*   **Intuitive Category Grouping:** Automatically categorizes your JSON keys into *Meta, Content, Media,* or *Values* groups based on naming patterns.
*   **Robust Undo Engine:** State tracking up to 20 operations deep with visual delta/differential highlighting so you can see exactly what changed during a reversion.
*   **Seamless I/O:** Remembers your URL endpoint history locally and exports perfectly structured, cleaned JSON files instantly.

## 🛠️ How to Use Locally
1. Clone this repository or download the `index.html` file.
2. Double-click `index.html` to open it in any modern browser.
3. Input your JSON URL (or historical shortcut), apply filters, make your structural updates, and click **Download JSON**.
